# aberdeen-traffic-collisions-heatmap
Plot a heatmap of traffic accidents using the datasets provided by ODIaberdeen. This project is a work in progress but you can preview it's functionality [here](https://telekineticyeti.github.io/aberdeen-traffic-collisions-heatmap/).

### To Compile
```ng build --prod --aot --no-sourcemaps --buildOptimizer -cc -vc -dop```
